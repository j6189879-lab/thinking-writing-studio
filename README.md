# 思维写作馆

一个思维表达能力训练系统。不是日记，不是 AI 代写——而是每天 15 分钟，通过结构化写作练习，把模糊的想法变成清晰的文字。

## 设计理念

写作是思考的外化。你需要的不是更华丽的辞藻，而是更清晰的思维。思维写作馆提供六大类写作主题、AI 教练点评和成长追踪，帮你建立持续写作的习惯。

## 功能

- **📝 今日主题** — 每天自动推送一个写作主题，支持换题、随机抽签、AI 出题
- **🎴 六大主题卡牌** — 观察卡、回忆卡、观点卡、心理卡、思辨卡、成长卡，共 48 个内置主题
- **✍️ 专注写作** — 极简编辑器，实时字数统计，自动保存草稿
- **🤖 AI 点评** — 「墨老师」写作教练点评（清晰度 / 具体度 / 逻辑性 / 深度），不配置 API 时自动降级为本地模拟点评
- **📈 成长记录** — 能力维度雷达图、每日字数统计、历史文章回顾
- **🏆 等级系统** — 10 级成长体系，从「观察者」到「思维创作者」

## 使用方法

直接双击 `index.html` 用浏览器打开即可，无需安装任何东西。

所有数据保存在浏览器本地（localStorage），不会上传到任何服务器。

## 可选配置

如需使用真实 AI 点评和 AI 出题功能，打开设置页面，配置：

- **API Base URL**：DeepSeek 填 `https://api.deepseek.com/v1`，通义千问填 `https://dashscope.aliyuncs.com/compatible-mode/v1`
- **API Key**：你的 API 密钥
- **模型名称**：如 `deepseek-chat`、`qwen-turbo`

不配置也能正常使用，系统会自动用本地规则评分。

## 数据导出

设置页面提供「导出数据」按钮，可将所有文章和进度导出为 JSON 文件备份。

## 技术栈

| 层级 | 技术 |
|------|------|
| 页面 | 原生 HTML + CSS |
| 逻辑 | 原生 JavaScript（无框架） |
| 存储 | localStorage |
| AI | OpenAI 兼容 API 格式 |
| 字体 | Noto Sans SC + Noto Serif SC |
